# Space Invader

Proyecto de desarrollo de un videojuego inspirado en el clásico **Space Invaders**, realizado con fines académicos.

## 🎮 Descripción
El proyecto consiste en la implementación de un juego donde el jugador controla una nave espacial y debe eliminar oleadas de enemigos, evitando sus ataques.  
El desarrollo sigue una estructura modular y está completamente documentado mediante una wiki técnica.

## 🎯 Objetivos
- Recrear el videojuego Space Invaders.
- Aplicar buenas prácticas de programación.
- Diseñar una interfaz clara e intuitiva.
- Documentar el proyecto de forma estructurada.
